# Presentación sugerida: AnimeVision

## Diapositiva 1. Portada
- AnimeVision
- Recomendador de anime por imagen con IA
- Integrantes
- Materia

## Diapositiva 2. Problema
- Las recomendaciones tradicionales dependen de búsquedas manuales.
- Muchas veces el usuario solo conoce una imagen o una estética.
- No siempre sabe cómo describir el tipo de anime que busca.

## Diapositiva 3. Solución
- Subir una imagen.
- Analizarla con IA.
- Traducir el resultado a géneros y etiquetas.
- Consultar AniList y mostrar animes similares.

## Diapositiva 4. Requerimientos cumplidos
- Front-end
- Back-end
- Inteligencia artificial
- Login
- Despliegue
- README con guía
- Reporte y presentación en GitHub

## Diapositiva 5. Arquitectura
- Front-end React
- Back-end Express
- Base de datos MongoDB Atlas
- Servicio de IA con FastAPI + CLIP
- API externa AniList

## Diapositiva 6. Flujo
1. Usuario inicia sesión.
2. Sube una imagen.
3. La IA detecta etiquetas.
4. El back-end consulta AniList.
5. Se muestran recomendaciones.
6. Se guarda historial y favoritos.
